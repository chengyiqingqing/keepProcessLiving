# 无人机问题
操作说明

输入输出用例如下：

	请做第一项输入：记录无人机活动信号的文本内容（输入end时，将代表第一项输入结束）
	plant1 1 1 1
	plant1 1 1 1 1 2 3
	plant1 2 3 4 1 1 1
	plant1 3 4 5
	plant1 1 1 1 1 2 3
	end
	请做第二次输入：（输入exit时，将代表第二项输入结束，退出程序）
	0	（输入）
	plant1 0 1 1 1	(输出)
	1	（输入）
	plant1 1 2 3 4 （输出）
	2	（输入）
	plant1 2 3 4 5 (输出)
	3 	（输入）
	error : 3	   (输出)
	4 	（输入）
	error : 3	   (输出)
	5	（输入）	
	cannot find 5  (输出)
